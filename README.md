# ⚡ SPPG GABUT 404

> **Yayasan Ngopi Dulu Baru Kerja**

Dashboard admin bergaya futuristik sci-fi dengan tema cyberpunk, efek visual canggih, dan fitur interaktif.

## ✨ Fitur

- 🔐 **Sistem Login Hacker-Style** — Halaman autentikasi dengan efek terminal (Kode akses: `SPPG2026`)
- 🌗 **Light / Dark Mode** — Toggle tema terang dan gelap secara real-time
- 🕐 **Jam Digital HUD** — Jam real-time dengan presisi milidetik di navbar
- 🔊 **Efek Suara Sci-Fi** — Suara sintetis saat hover dan klik tombol (Web Audio API)
- 📊 **Grafik Statistik** — Area chart interaktif menggunakan Recharts
- 🖥️ **Terminal Log** — Simulasi log aktivitas sistem secara real-time
- ✨ **Partikel & Animasi** — Background partikel interaktif, efek glitch, scanline, dan glassmorphism

## 🛠️ Tech Stack

- **React** + **Vite**
- **Tailwind CSS v4**
- **Framer Motion** — Animasi halaman dan komponen
- **Recharts** — Grafik dan visualisasi data
- **tsParticles** — Background partikel interaktif
- **Lucide React** — Ikon modern
- **Web Audio API** — Efek suara tanpa file eksternal

## 🚀 Cara Menjalankan

```bash
# Clone repository
git clone https://github.com/Abibsa/sppg-gabut-404.git
cd sppg-gabut-404

# Install dependencies
npm install

# Jalankan dev server
npm run dev

# Build untuk production
npm run build
```

## 📁 Struktur Folder

```
src/
├── components/       # Komponen UI (Navbar, MenuButton, dll)
├── config/           # Konfigurasi situs & menu
├── context/          # React Context (Theme & Auth)
├── pages/            # Halaman (Home, Arsip, Login)
├── styles/           # CSS global & tema
└── utils/            # Utilitas (Sound Effects)
```
